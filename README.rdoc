= Copy Fixer (Firefox add-on)

Ctrl-C/Command-C copy page title and URL when select nothing.

何も選択してない状態で Ctrl-C / Command-C を押すと、ページタイトルとURLをコピーするようになるFirefox add-onです。

right click and download. doble click. 右クリックでダウンロードして、ダブルクリックでインストールしてください。
